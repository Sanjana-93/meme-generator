# meme-generator

A simple and interactive web application that generates random memes using an API. Users can generate new memes with a single click and enjoy trending meme templates instantly.

🌐 Live Demo:
https://sanjana-meme-generator.netlify.app/

Features:- 

1. Generate random memes
2. Fetch memes dynamically from API
3. Fast and responsive UI
4. Mobile-friendly design
5. Simple and user-friendly interface

Tech Stack:- 

   HTML,
   CSS,
   JavaScript,
   Public Meme API.
   Netlify (Deployment)

How to Run Locally:-

1. Clone the repository
2. git clone https://github.com/your-username/meme-generator.git
3. Open the project folder
4. Open index.html in your browser

Concepts Used:-

 1. DOM Manipulation
 2. Fetch API
 3.Asynchronous JavaScript (Promises)
 4. Event Handling
 5. API Integration
