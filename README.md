# meme-generator

A simple and interactive web application that generates random memes using an API. Users can generate new memes with a single click and enjoy trending meme templates instantly.

🌐 Live Demo:
https://sanjana-meme-generator.netlify.app/

Features:- 

1. Generate random memes
2. Fetch memes dynamically from API
3. Fast and responsive UI
4. Mobile-friendly design
5. Simple and user-friendly interface

Tech Stack:- 
   HTML
   CSS
   JavaScript
   Public Meme API
   Netlify (Deployment)
